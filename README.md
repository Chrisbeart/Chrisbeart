### What I Build

**AI Agents & Intelligent Automation**
> Custom AI agents for enterprise workflows - automated ticket triage, intelligent routing, conversational interfaces, and decision support systems using Atlassian Forge/Rovo, LangChain, and LLM APIs.

**Cloud Infrastructure**
> Production-grade cloud architectures on AWS and Azure. Infrastructure as Code with Terraform, configuration management with Ansible, and fully automated CI/CD pipelines.

**Enterprise Applications**
> Full-stack applications for healthcare, logistics, and business operations. React frontends, Node.js/Python backends, and robust database designs handling real-world data at scale.

**ERP & System Integration**
> Integration solutions connecting ERP systems, EDI partners, and business APIs. Automated data flows between enterprise systems.
